# Sound Files

Place your custom WAV sound files here:

- `start.wav` — Played when recording starts
- `stop.wav` — Played when recording stops

Requirements:
- Format: WAV (16-bit, 44.1kHz)
- Duration: < 0.5 seconds
- Volume: Moderate (will respect system volume)

If no custom sounds are provided, YapYap will use the system beep sound.
